Emergency Access

When your agent breaks at 2 AM

1. SSH from your phone

Termius app → SSH to your Tailscale IP → log in as root

2. Use a non-autonomous coding agent on root

Copilot CLI, Claude Code, or opencode — installed on root, not the openclaw user

$ copilot-cli

> "OpenClaw is stuck in a loop. Check the logs and fix it."

3. The pattern

OpenClaw tells you the error → you paste it into the coding agent → it debugs and fixes

You’re using an agent to fix an agent. Welcome to 2026.
