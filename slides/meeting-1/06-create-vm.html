Create the VM

Azure CLI — two commands

# 1. Resource group

az group create --name personal-agents --location westus2

# 2. Virtual machine

az vm create \

--resource-group personal-agents \

--name openclaw-vm \

--image Ubuntu2204 \

--size Standard_B2ms \

--admin-username azureuser \

--generate-ssh-keys

B2ms = 2 vCPU, 8GB RAM — ~$60/mo — well within your $150 free credits

⚠️ This creates a public IP. Port 22 is open to the internet right now.
