ClawCamp Setup

Bootcamp-style guide from AI Daily Brief. Organized into modules. Point agent at it — it iteratively configures itself via Edge TTS voice.

USER.md

Name, timezone, preferences

AGENTS.md

Operating instructions

SOUL.md

Personality, guardrails

⚠️ Security Caveat

The onboarding wizard installs the CLI tool, but my setup intentionally does NOT give OpenClaw access to its own API keys. Deliberate friction — the agent can’t use its own keys to install or modify things.

The more context it has, the less dumb it acts.
