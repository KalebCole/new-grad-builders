Security

I’m not a security engineer — but I didn’t want to be low-hanging fruit

🔒 SSH Hardening

No root login, key-only auth, 3 max attempts, modern crypto only

🔥 UFW Firewall

Deny all inbound except SSH. After Tailscale: SSH from VPN only

🛡️ fail2ban

Auto-bans IPs after failed SSH attempts

☁️ Azure Metadata Lockdown

Only root can query the Instance Metadata Service (blocks SSRF)

👤 User Isolation

openclaw user: no sudo, no docker. Blast radius containment.

🔐 systemd LoadCredential

Secrets on root (chmod 600). Injected via tmpfs at runtime. Never on disk for the agent.

🏰 Process Sandboxing

NoNewPrivileges, ProtectSystem=strict, PrivateTmp, restricted syscalls

🌐 Tailscale

WireGuard mesh VPN. No public SSH needed once connected.

This is my implementation. Poke holes in it.
